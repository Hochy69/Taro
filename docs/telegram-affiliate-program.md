## 3. Как включить программу (владельцу бота)

> API-метод `bots.updateStarRefProgram` **недоступен через Bot Token** — только через приложение Telegram или MTProto (аккаунт владельца). См. `scripts/setup_telegram_affiliate.py`.

### Способ A — через Telegram (рекомендуется)

1. Откройте **@BotFather** → ваш бот **@best1tarolog_bot**.
2. Убедитесь, что привязан **Mini App** (Web App URL).
3. Откройте **профиль бота** в Telegram (не BotFather) → **Изменить** → **Affiliate program** / **Партнёрская программа**.
4. Укажите:
   - **Commission:** `25%` (250 permille)
   - **Duration:** `3 months`
5. Сохраните.

Если пункта нет — обновите Telegram до последней версии; нужен бот с Mini App и правами владельца.

### Способ B — скрипт Telethon (опционально)

```bash
pip install telethon
export TELEGRAM_API_ID=...
export TELEGRAM_API_HASH=...
python scripts/setup_telegram_affiliate.py
```

При первом запуске — вход по номеру телефона владельца аккаунта.
